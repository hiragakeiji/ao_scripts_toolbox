# ao_scripts_toolbox

Small utility scripts for DCC tools and pipeline workflows.
Focus: simple distribution (no build), practical tools, clear usage.

## Contents
- [Tool Index](docs/index.md)
- [Conventions](docs/conventions.md)

## Supported
- Maya: 2025 (Python / PySide6 tools may exist per script)
- Houdini: (write versions per tool)
- Windows: PowerShell / Batch
- Python: small CLI / utilities

## How to use
Each tool has its own folder with:
- `README.md` (usage)
- the script file(s)

Start from the tool index: `docs/index.md`.
